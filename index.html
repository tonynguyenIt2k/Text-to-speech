Tài liệu hướng dẫn API

Tích hợp dịch vụ CapCut TTS & STT trực tiếp vào ứng dụng của bạn.

Cấu hình kết nối API Server

Nhập địa chỉ API Server (ví dụ: http://localhost:8000). Để trống nếu chạy cùng server.

1. Chuyển văn bản thành giọng nói (TTS)

POST /api/tts

JSON Payload:

{
  "text": "Xin chào thế giới",
  "voice": "BV074_streaming",
  "rate": 1.0
}

Fetch API (JavaScript):

fetch('http://127.0.0.1:8000/api/tts', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({ text: "Xin chào", voice: "BV074_streaming" })
})
.then(res => res.json())
.then(data => console.log("Speech URL:", data.speech_url));

2. Nhận diện giọng nói (STT)

POST /api/stt

Sử dụng multipart/form-data.

Python requests:

import requests

url = "http://127.0.0.1:8000/api/stt"
files = {'file': open('audio.mp3', 'rb')}
data = {'language': 'vi-VN', 'use_translation': 'false'}

response = requests.post(url, files=files, data=data)
for u in response.json().get('utterances', []):
    print(f"[{u['start_time']}ms] {u['text']}")
