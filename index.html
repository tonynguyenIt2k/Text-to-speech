Tài liệu hướng dẫn API

Tích hợp dịch vụ CapCut TTS & STT trực tiếp vào ứng dụng của bạn một cách nhanh chóng.

Cấu hình kết nối API Server

Nhập địa chỉ API Server của bạn (ví dụ: http://localhost:8000 hoặc link máy chủ cloud). Để trống nếu chạy trực tiếp trên cùng một máy chủ backend.

1. Chuyển văn bản thành giọng nói (TTS)

POST /api/tts

Cấu trúc yêu cầu (JSON Payload):

{
  "text": "Xin chào thế giới",
  "voice": "BV074_streaming",
  "rate": 1.0 // Tốc độ từ 0.5 đến 3.0
}

Mẫu Fetch API (JavaScript):

fetch('http://127.0.0.1:8000/api/tts', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({
    text: "Xin chào thế giới",
    voice: "BV074_streaming"
  })
})
.then(res => res.json())
.then(data => console.log("Speech URL:", data.speech_url));

2. Nhận diện giọng nói (STT)

POST /api/stt

Sử dụng định dạng truyền tải multipart/form-data để tải tệp tin lên.

Mẫu tích hợp (Python requests):

import requests

url = "http://127.0.0.1:8000/api/stt"
files = {'file': open('audio.mp3', 'rb')}
data = {
    'language': 'vi-VN',
    'use_translation': 'false'
}

response = requests.post(url, files=files, data=data)
result = response.json()
for utter in result.get('utterances', []):
    print(f"[{utter['start_time']}ms -> {utter['end_time']}ms] {utter['text']}")
